# Sales Data Cleaning with Pandas

This project simulates a real-world ETL process for sales data.

## Dataset

The generated dataset intentionally contains:

- Missing values
- Duplicate purchase IDs
- Invalid dates
- Mixed currency formats
- Incorrect profit values
- Different payment methods
- Inconsistent sales channels

## Cleaning Process

- Remove duplicate purchase IDs
- Standardize monetary values
- Convert dates
- Handle missing values
- Normalize categorical columns
- Recalculate profit
- Export a clean dataset

## Technologies

- Python
- Pandas
